Cómo funcionan las alertas
• El navegador consulta la API cada intervalo y compara las posiciones con la lectura anterior.
• On-chain solo se puede detectar la entrada en el momento en que se ejecuta (segundos), no antes.
• Los avisos del navegador (🔔) solo suenan con esta pestaña abierta. Para alertas 24/7 a Telegram/Discord aunque tu PC esté apagado, hace falta un backend (GitHub Actions cron) — pídemelo y te lo genero.
